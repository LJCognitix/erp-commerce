# New Directory Structure

This is a professional reorganization of the `src` directory structure for better maintainability and clarity.

## Directory Breakdown

1. `src/hooks/` - Custom React hooks
2. `src/services/` - API client and services
3. `src/schemas/` - Zod validation schemas by domain
4. `src/middleware/` - Authentication and permissions middleware
5. `src/lib/` - Core utility files

Each folder contains relevant files to keep the project organized and maintainable.
